
Q-learning is a model-free, off-policy reinforcement learning algorithm that enables an agent to learn the optimal action-selection policy by interacting with an environment and maximizing cumulative rewards. It uses a Q-table to store the expected future value of state-action pairs, updating them iteratively based on rewards received.
